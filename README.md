# Taller Colaborativo - Programa de Funciones Matemáticas

##  Descripción
Programa interactivo en Python con menú de funciones matemáticas desarrollado colaborativamente usando Git y GitHub.

##  Funcionalidades

1. **Fibonacci**: Calcula serie de Fibonacci
2. **Factorial**: Calcula factorial con visualización
3. **Números Primos**: Verifica si un número es primo
4. **Números Perfectos**: Genera números perfectos

##  Ejecución
```bash
python main.py
```

```
taller-colaborativo-python/
```

##  Metodología Git

- Rama principal: `main`
- Ramas de funcionalidades: `menu`, `feature-fibonacci`, `feature-factorial`, `feature-primos`, `feature-perfectos`
- Pull Requests para cada funcionalidad
- Merge secuencial

## Aprendizajes

- Trabajo colaborativo con Git
- Branching y merging
- Pull Requests
- Code review
